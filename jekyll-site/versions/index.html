1.1.0 Released on 2/4/2014

Set the version of the project in the node properties, also add extra properties (as the cross compiled version) when adding a new project. If the node exists already (maybe it is a dependency of another project), no extra nodes will be created, instead the same node will be matched, adding the properties. This is done using the cypher syntax MERGE - ON CREATE - ON MATCH
